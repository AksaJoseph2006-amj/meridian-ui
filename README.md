# 🌍 Meridian

A modern and responsive travel website UI built with **HTML, CSS, JavaScript, and GSAP**. Meridian is a front-end concept that showcases an immersive landing page with smooth animations, engaging visuals, and a clean user experience inspired by modern travel websites.


## 🚀 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* GSAP (GreenSock Animation Platform)



## ✨ Features

* Responsive design for different screen sizes
* Smooth page animations powered by GSAP
* Modern travel-inspired UI
* Interactive navigation
* Animated hero section
* Clean and minimal layout
* Organized and maintainable code structure



## 👤 What Users Can Do

* Explore the travel-themed landing page
* Navigate through different sections
* Experience smooth animations and transitions
* View a responsive interface across devices

> **Note:** This project is a front-end UI concept and does not include backend functionality or booking features.



## 🛠️ How I Built It

I started by designing the overall layout using semantic HTML and styled it with CSS to create a clean, modern interface. JavaScript was used to add interactivity, while GSAP was integrated to create smooth entrance animations, transitions, and scrolling effects that enhance the overall user experience.



## 📚 What I Learned

* Creating engaging animations with GSAP
* Structuring a responsive landing page
* Improving JavaScript DOM manipulation skills
* Building smoother user experiences through motion design
* Organizing front-end projects with better code structure
